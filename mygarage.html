Overview

About the project

A graduation project built to solve the real problem of disorganized garage booking in Bahrain using modern web technologies.

01

The problem

Car owners in Bahrain face difficulties finding trusted garages. Information is scattered across Instagram, WhatsApp, and personal recommendations. Garage owners lack tools to manage bookings and payments efficiently.

02

The solution

My Garage provides one centralized platform connecting customers, garage owners, and administrators. Customers browse, book, track, and pay — all in one place. Garage owners get a full management dashboard.

03

AI assistant

An integrated AI Garage Assistant provides users with guidance about common vehicle problems and maintenance issues — improving the user experience without requiring human support at all times.

04

Tech foundation

Built using HTML, CSS, JavaScript, Firebase Authentication, and Cloud Firestore — supporting secure login, real-time database operations, and a responsive interface accessible from any device.
